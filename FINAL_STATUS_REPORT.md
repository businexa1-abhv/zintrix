## 🚀 How to Deploy

### 1. **Local Development**
```bash
cd zintrix
npm install
npm run dev
```

### 2. **Production Build**
```bash
npm run build
npm run preview
```

### 3. **Deploy to Hosting**
Options:
- Vercel (Recommended for Vite)
- Netlify
- GitHub Pages
- Traditional hosting

### 4. **Configure WhatsApp**
- Update phone number in `Contact.jsx` and `WhatsAppButton.jsx`
- Test on desktop and mobile
- Setup WhatsApp auto-reply
