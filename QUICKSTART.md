## 🚀 Getting Started

### Install & Run
```bash
cd zintrix
npm install
npm run dev
# Open http://localhost:5173
```

### Deploy
```bash
npm run build
# Upload dist/ to hosting
# Or use: vercel / netlify deploy
```
