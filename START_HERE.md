## 🚀 How to Get Started

### 1. Install Dependencies
```bash
cd c:\Users\DELL\Desktop\worksupport\zintrix-landing-page\zintrix
npm install
```

### 2. Run Development Server
```bash
npm run dev
# Opens at http://localhost:5173
```

### 3. Customize Your Data
See `CUSTOMIZATION_CHECKLIST.md` for:
- Update testimonials
- Set your pricing
- Add team members
- Customize FAQ
- Update contact info

### 4. Build for Production
```bash
npm run build
# Creates dist/ folder for deployment
```

### 5. Deploy
```bash
# Option A: Vercel (easiest)
npm i -g vercel && vercel

# Option B: Netlify
npm i -g netlify-cli && netlify deploy --prod --dir=dist

# Option C: Manual upload dist/ folder to any hosting
```
